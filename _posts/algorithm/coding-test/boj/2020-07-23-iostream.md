## 🔔 11021 ⭕

> 문제 👉 <https://www.acmicpc.net/problem/11021>

### 내 풀이 

```cpp
#include <stdio.h>
using namespace std;

int main()
{
   int n, a, b;
    
   scanf("%d", &n);
   
   for(int i = 1; i <= n; i++)
   {
       scanf("%d %d", &a, &b);
       printf("Case #%d: %d\n", i, a + b);
   }
   
   return 0;
}
```

- `scanf("%d %d", &a, &b);`
  - "1 3" 혹은 "1,3" 이런식으로 어떤 형식에 따라 입력을 받을 땐 `scanf`쓰는게 편한 것 같다. 
    - 어떤 형식에 따라 입출력 해야할 땐 `<stdio>` 인클루딩한 후 `scanf`와 `printf`를 사용하자.
